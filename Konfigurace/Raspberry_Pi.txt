Hostname prvak-pi.

Custom unit soubory pro systemd v [[/etc/systemd.]]

Vnejsi rozhrani: eth0, vnitrni rozhrani: eth1

Nejdriv se musi zapnout eth1 a nastavit IP adresa.
	ip link set up dev eth1
	ip addr add 10.0.0.1/8 dev eth1
	
[[/etc/sysctl.d/30-ipforward.conf]] nastavi IP forwarding.
	# /etc/sysctl.d/30-ipforward.conf:
	# net.ipv4.ip_forward=1
	# net.ipv6.conf.default.forwarding=1
	# net.ipv6.conf.all.forwarding=1
	
	sysctl net.ipv4.ip_forward=1
	
iptables konfigurace zapne maskaradu.
	iptables -t nat -A POSTROUTING -o eth0 -j MASQUERADE
	
SSH demon posloucha jenom vevnitr.
DHCP server by mel poslouchat jenom na vnitrnim rozhrani — je to nastavene v jeho unitu (v [[/etc/systemd]] — jsou tam overridy)
